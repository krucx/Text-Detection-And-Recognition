0. Download Git and Heroku CLI
1. Download this github repository 
2. Open terminal in web app folder
3. Add your model in flaskpackage/static folder(refer to instruction.txt in static folder)
4. Run command 'git init' 
5. Run command 'git add .'
6. Run command 'git commit -m "initial commit" '
7. Run command 'heroku create'
8. Run command 'heroku stack:set heroku-18'
9. Run command 'git push heroku master '
